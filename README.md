# pdf-notebook-viewer
A modern, lightweight PDF viewer designed for **interactive reading and note-taking**.  
Perfect for students, researchers, and developers who want a seamless way to read PDFs alongside notes.

##  Features

-  **Smooth PDF Rendering**
  - Fast and responsive document viewing
  - Supports multi-page navigation

- **Notebook-Style Interface**
  - Take notes alongside your PDF
  - Structured reading + writing workflow

- **Zoom & Navigation**
  - Zoom in/out for better readability
  - Quick page navigation controls

- **Lightweight & Fast**
  - Minimal dependencies
  - Optimized for performance

-  **User-Friendly UI**
  - Clean and distraction-free design
  - Focused on productivity

## Tech Stack
Frontend: HTML, node.js, react
PDF Engine: PDF.js
Styling: Tailwand CSS
